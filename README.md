## ☕ Smokey Coffee - Café Website
A modern, responsive multi-page website for an artisan coffee shop.

📁 Files
File	Description
index.html	Homepage (products, reviews, brewing tips)
menu.html	Full menu (coffee, pastries, savory, seasonal)
about.html	Our story, team, sustainability
contact.html	Contact form, map, FAQ
🚀 Quick Start
Download all 4 HTML files

Put them in the same folder

Open index.html in your browser

🛠️ Built With
HTML5 + Tailwind CSS

Font Awesome icons

Google Fonts (Inter)

📱 Features
Fully responsive (works on phone/tablet/desktop)

Centered navigation bar

Coffee cup favicon

Hover effects & smooth transitions

🔗 Navigation
Home → Menu → Our Story → Contact (all pages linked)
